# 🚦 Traffic Light Assistant for Visually Impaired

YOLO11とOpenCVを組み合わせた、視覚障害者支援のための信号機色判別アプリです。

## ✨ 機能
- **YOLO11**: 画像内の信号機をリアルタイムで特定。
- **OpenCV**: 特定された信号機の色をHSV解析で高精度に判別（日本の信号機の色味に対応）。
- **Streamlit**: スマホのブラウザから簡単にアクセス可能。

## 🚀 使い方
1. リポジトリをクローンまたはダウンロード。
