# 🚦 信号機判別支援アプリ

YOLO11とOpenCVを組み合わせた、**視覚障害者支援のための信号機色判別アプリ**です。  
撮影するだけで信号の色を判定し、**音声で結果を読み上げ**ます。

---

## ✨ 機能

- **YOLO11**: 画像内の信号機をリアルタイムで検出
- **OpenCV**: HSV解析で信号機の色を高精度に判別（日本の信号機の色味に対応）
- **音声読み上げ**: 判定結果をブラウザの音声合成APIで自動読み上げ（日本語対応）
- **アクセシビリティ対応UI**: 大きな文字・高コントラスト・シンプルなレイアウト
- **Streamlit**: スマホのブラウザから簡単にアクセス可能

---

## 🔊 音声読み上げ内容

| 判定結果 | 読み上げ |
|----------|----------|
| 青信号 | 「青信号です。進めます。」 |
| 赤信号 | 「赤信号です。止まってください。」 |
| 黄信号 | 「黄信号です。注意してください。」 |
| 未検出 | 「信号機が見つかりませんでした。正面から大きく写してください。」 |

---

## 🚀 使い方

1. アプリをブラウザで開く
2. 「📸 信号機を撮影してください」ボタンを押す
3. 信号機を正面から撮影する
4. 判定結果が画面に大きく表示され、音声で読み上げられる

---

## 🛠 ローカルでの実行方法

```bash
# リポジトリをクローン
git clone https://github.com/densetunotanaka15-cmd/shiny-broccoli.git
cd shiny-broccoli

# 依存ライブラリをインストール
pip install -r requirements.txt

# アプリを起動
streamlit run app.py
```

## 📦 使用技術

| ライブラリ | 用途 |
|------------|------|
| [Streamlit](https://streamlit.io/) | WebアプリUI |
| [Ultralytics YOLO11](https://github.com/ultralytics/ultralytics) | 信号機検出 |
| [OpenCV](https://opencv.org/) | 色判定（HSV解析） |
| [Pillow](https://pillow.readthedocs.io/) | 画像処理 |

## ⚠️ 注意事項

本アプリは補助的なツールです。  
必ず周囲の音・誘導鈴・歩行者用信号の音を確認して安全を確保してください。
